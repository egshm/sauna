# 架空のサウナ施設紹介サイト「KOHAN SAUNA」

架空のサウナ施設「KOHAN SAUNA」の紹介サイトで、トップページ1枚です。  
もともとは、Photoshopデザインカンプからコーディングした、架空建設会社のコーポレートサイトでしたが、  
そこからテキストや画像を差し替えて、半オリジナル化しました。レイアウトはほとんど変えていません。  
私はサウナが好きなのですが、大自然とサウナの融合という、自分にとっての理想を素直に表現してみました。  

## デザインツール
- Photoshop

## サイト構成
トップページのみ

## 使用技術
- HTML5
- CSS / Sass (Dart Sass)
- JavaScript (ES6+)
- jQuery 3.7.1
- slick 1.8.1

## CSS設計
- BEM + FLOCSS

## 実装機能・UIパーツ
- ハンバーガーメニュー
- ローディング画面
- スクロールスパイ（現在位置に応じたナビゲーションのハイライト）
- スムーススクロール
- ページの先頭に戻るボタン
- スライダー（slick.js）

## WAI-ARIA
まだまだ勉強中ですが、以下の点を意識して実装しています。

- VSCodeの拡張機能`markuplint`と使ってWAI-ARIAの構文チェックをしながら実装
- セマンティックなHTML要素を使用
- 装飾目的だけの`svg`タグや`img`タグには`aria-hidden="true"`を適用
- ハンバーガーメニューは、アイコンとメニューを`aria-controls`属性で関連付け、メニューが展開されているかどうかを`aria-expanded`属性で表現
